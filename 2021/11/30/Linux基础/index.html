  • Linux基础

一、Linux

1.概念

  • 内核
    • 只是提供操作系统的基本功能，内存管理、进程管理
  • 发行版
    • 基于linus维护linux内核，桌面应用和命令行系统应用
  • 常见发行版
    • RedHat中国和北美地区流行
    • CentOs 社区版本 免费
    • 乌班图Ubuntu

2.Linux和Windwos区别

  • 大小写严格区分
  • 后缀不同
    • 安装文件exe ==> rpm
    • 压缩包 zip ==> tar.gz

3.使用字符界面的好处

  • 占用系统资源非常少

4.程序员是如果使用Linux的

环境分类

  • 开发环境：编码环境
  • 测试环境：和开发环境基本一致
  • 生产环境：给客户用的真实环境

项目环境部署：Linux

服务器类型：自建服务器，租用服务器

  • 使用SSH远程连接
    • SSH为建立在应用层基础上的安全协议。SSH是较可靠的，专为远程登录会话和其他网络服务提供安全性的协议。

二、Linux基础操作

1.Linux基础命令

  • linux基础目录
    • ~表示当前所在的目录
    • #表示的是超级用户
    • $表示普通用户

  • 查看当前目录下所有文件
    • ls -a
  • 切换目录
    • cd ~ 切换到家目录
    • cd ：切换用户目录
    • cd - ：切换到上一目录
    • cd.. ：切换到上一个目录
    • cd . ：切换到当前目录
  • 查看路径
    • pwd
  • 创建文件
    • touch 文件名
  • 创建文件夹
    • mkdir
    • mkdir -p xx/yy/zz
      • -p 递归创建多级目录
  • 删除
    • rmdir
    • rm
      • -r 递归删除目录
      • -f 强制删除
      • 禁止使用：rm -rf /*

  • 复制

    • cp [选项] [原文件或目录] [目标]
      • -r 复制目录

  • 剪切或改名

    • mv [原文件或目录] [目标目录]

  • 环境变量

    • PATH echo $Path

  • 压缩

    • zip [压缩后] zip [被压缩]

    • 默认使用国外服务器进行下载
    • yum install zip

  • 解压

    • unzip [压缩文件]

  • 更换下载镜像地址为了阿里云

  • 当你有大段的空闲时间并且网络稳定的情况下，执行以下命令，更新所有软件
    • yum -y update
  • 打包
    • tar -cvf
      • -c 打包
      • -v 显示过程
      • -f 指定打包后文件名
    • tar -xvf 打包文件名
    • tar -zcvf aaa.tar.gz [要压缩的文件] 压缩文件
    • tar -zxvf aaa.tar.gz -C ./ 解压到当前目录

7.关闭防火墙

  • 关闭当前防火墙

    • systemctl stop firewalld

  • 永久关闭防火墙

    • systemctl disable firewalld

  • 给防火墙开口子

    • firewall-cmd –zone=public –add-port=8080/tcp
    • firewall-cmd –zone=public –add-port=80/tcp –permanent

  • 重新加载防火墙

    • firewall-cmd –reload

8.安装mysql

  • 上传Mysql文件

  • 卸载，防止冲突

    • rpm -e postfix mariadb-libs

  • 安装mysql依赖net-tools perl

    • yum -y install net-tools perl

  • 安装mysql-common

    • rpm -ivh mysql-community-common-5.7.22-1.el7.x86_64.rpm

  • 安装mysql-libs

    • rpm -ivh mysql-community-libs-5.7.22-1.el7.x86_64.rpm

  • 安装mysql-client

    • rpm -ivh mysql-community-client-5.7.22-1.el7.x86_64.rpm

  • 安装mysql-server

    • rpm -ivh mysql-community-server-5.7.22-1.el7.x86_64.rpm

  • 设置开机自启动

    • systemctl enable mysqld

  • 启动mysql

    • systemctl start mysqld

  • 设置mysql密码

    • 返回根目录
    • vi /etc/my.cnf
