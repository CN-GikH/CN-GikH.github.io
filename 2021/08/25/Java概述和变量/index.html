五、Java的数据类型

  • 每一个数据类型的存储范围都不一样，为了提高内存的使用率

1.整数

整数 长度
byte 8bit
short 16bit
int(默认) 32bit
long 64bit
  • 在Java中，出现的所有整数，默认都是int类型
  • byte在存整数时，如果int类型整数在byte范围内，byte会自动完成转换 int—byte

2.小数

小数 长度
float 浮点型 32bit
double(默认) 双精度 64bit
  • 长整型long后缀使用L标记，表明是长整型
  • 单精度float后缀使用F标记，表名是单精度

七、进制

1.什么是进制？

  • 逢几进一

2.Java常见的进制

  • 二进制：0b
  • 八进制：0
  • 十进制：默认
  • 十六进制：0x

3.进制间的转换

  • 任意进制到十进制

    • 系数 * 基数的权次幂相加

      • 系数：位数是几
      • 基数：原来的进制
      • 权：从个位开始，从0向左依次增加

  • 十进制到任意进制

    • 除以基数，倒取余数

  • 二进制 <==>八进制

    • 421码

  • 二进制 <==> 十六进制

    • 8421码

4.源码、反码、补码

  • 在计算机中，二进制的运算，是以补码的形式进行的(0为正，1为负)

  • 正数：源码 = 反码 = 补码

  • 负数：源码不变

    ​ 反码：符号位不变，其余按位取反

    ​ 补码：反码+1

八、运算符

1.什么是运算符

  • 运算符是对一个、两个或三个操作数执行特定操作，然后返回结果的特殊符号。

2.运算符的分类

①算数运算符
算数运算符 效果
+ 、 - 加、减
* 、/ 乘、除
% 取模（即取余）
②一元运算符
  • 只需要一个操作数，就可以对其进行相应的操作
一元运算符 效果
+ 表示正值（可省略）
- 表示负值，即否定表达式
++ 将值+1
– 将值-1
! 反转布尔值

a++; ++b;

  • 单独使用时，++即自加一，–即自减一，放在前后效果相同

int b = a ++; int c = –b;

  • 参与运算时，放在前即先++或–， 放在后面则先运算再++、–
③比较运算符
  • 返回布尔类型的值，即true，false
比较运算符 效果
== 等于
!= 不等于
> 大于
< 小于
<= 小于等于
>= 大于等于
④条件运算符
  • 用来连接比较表达式（即布尔类型）表达式
逻辑运算符 作用
& AND(与) 有false则false，两个true则true
| OR(或) 有true则true，两个false则false
^ XOR(异或) 相同false，不同true
! NOT(非) 结论相反
&& AND(短路) 短路效果，即左边有false，右边不执行
|| OR(短路) 短路效果，即左边有true，右边不执行

Java中，若要表达3<x<6 ，应写为 x>3 & x<6
