数组

发表于 2021-08-25 分类于 Java 阅读次数： Valine：
学习java中数组的定义和使用，赋值取值等操作，一维数组和二维数组

一、数组

1.什么是数组

  • 概念：数组是存储多个变量的容器（数据类型要一致）
    • 数组可以存储基本数据类型也可以存储引用数据类型

2.定义数组

  • 第一种格式：

    • 数据类型 数组名[] ；

      例：int arr1[]；

  • 第二种格式：

    • 数据类型[] 数组名

      例：int[] arr2;

3.数组的定义和初始化格式

① 概念：
  • 从内存的原来看，数组是一组存储变量的容器
    • 并且内存中，堆内存是空闲的，所以我们可以在堆内存中定义一个数组存储数据
      • 要想在堆内存中创建数组，必须要将Cpu的执行权由栈区交给堆区（new）关键字
    • 在堆区中，开辟内存的目的是数据，而开辟内存空间，就需要用到数组类型
  • 以上两步可知，在堆区创建数组 new int[];
  • 创建数组后，需要在栈区创建一个变量，存储堆区数组的地址值，用来使用数组
②动态初始化：只指定长度，初始值默认
  • 系统初始默认值，int是0 ，String是null
③静态初始化：指定初始值，系统默认长度

4.数组的数据操作

Ⅰ.赋值操作
  • 数组名[索引] = 数组；
Ⅱ.取值操作

通过数组自带的索引取值

  • 直接取值
    • 直接打印到控制台，比较方便，但无法对数据继续进行操作
  • 赋值取值
    • 创建一个变量保存数据
      • 变量名 = 数组名[索引]
      • 比较繁琐，但可以对数据进行多次操作
Ⅲ.数组的特性
  • length表示数组的长度
    • arr.length
Ⅳ.练习
  • 数组中存5个整数，求和，以及求平均数
  • 数组中存5个数，找出最大、最小的两个数
  • 冒泡排序：从第一个数开始，依次和后一个数比较，小的放前，大的放后
Ⅴ.数组常见异常

  • 索引超出范围，索引越界

    • java.lang.ArrayIndexOutOfBoundsException:

  • 空指针异常

    • java.lang.NullPointerException

1.二维数组的概念

  • 二维数组就是一个元素为一维数组的数组

2.二维数组的定义格式

①动态初始化

arr[m] [n] 表示第m+1个一维数组的第n+1个数

没有给出一维数组的元素个数，可以动态给出

②静态初始化

3.二维数组的数据操作

Ⅰ.取数据

  • 直接取出

    • 用双层嵌套循环，第一层循环取出一维数组，第二层循环取出一维数组里的数

  • 赋值取
