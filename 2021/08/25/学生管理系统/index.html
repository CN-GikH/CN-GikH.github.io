一、面向对象思想

  • 暂且不提

二、数据类型

  • Java提供的数据类型有很多种，但是Java只提供了最基础的
  • 当需要使用自定义的数据类型时，Java提供了方法自行创建

1.自定义数据类型(定义类)

  • 通常叫做定义一个类
    • 实际上可以理解为自定义了一个数据类型

2.如何存储内部数据

  • 用定义变量的方式

3.自定义数据类型的使用

  • 使用思路

    • 通过内存图发现，我们需要用到自定义数据类型的内部数据
    • 而内部数据要存储数据就必须开辟内存空间
    • 遇到复杂类型的数据类型，需要在堆区中开辟内存空间
    • 将CPU执行权有栈区交到堆区new

4.自定义数据类型的思路

  • 第一步：创建项目，测试环境
  • Java没有自带学生数据，需要我们自定义一个数据类型
  • 自定义数据类型的格式
    • 位置：class之间并列关系
    • 内容：用变量存储数据
    • 格式：class 数据类型 {}
  • 在定义的数据类型中存储内部数据
    • 和之间定义变量、数组、集合等一样
  • 创建自定义的数据类型的变量，存储new Student()的地址值。
    • 因为Student内有许多变量，遇到复杂数据类型，在堆区中开辟内存空间所以 Student s = new Student();
  • 调用对应数据
    • 赋值
      • 变量名.内部变量名 = 数值；
    • 取值
      • 输出取值
        • System.out.println(s.id);
      • 赋值取值
        • String a = System.out.println(t.name);

6.注意事项

  • 从作用的角度判断Java中类的分类
    • 测试类：包含main方法的类（程序入口）
      • 也是一种自定义的数据类型，不过不用来做存储，而是做Java程序的测试
    • 基础类
  • 内部变量：属性
  • 自定义数据类型：先将小的数据进行封装
