匿名对象、内部类、代码块、和修饰符等

发表于 2021-08-25 分类于 Java Valine：
了解并学习匿名对象、内部类、代码块、包、修饰符、增强for循环等

一、匿名对象

1.什么是匿名对象

  • ​ 没有名字的对象
    • 即堆区的地址值，没有用变量来保存

2.匿名对象的使用

  • 快速访问（测试用）

  • 传参

二、内部类

1.什么是内部类

  • 在类中定义类
    • 通过实验，发现Java语法允许在类的内部定义类
  • 生成Class文件格式 ： 外部类$内部类

2.内部类区别

  • 成员内部类：类中方法外

    • 访问格式：

  • 局部内部类：类中方法中

    • 访问：
      • 在方法中，内部类并列的地方，创建内部类的对象，通过对象调用内部类中的成员

3.内部类访问特点

  • 内部类可以直接访问外部类的成员，包括私有
  • 外部类要访问内部类的成员，必须创建对象

4.匿名内部类

  • 快速的实例化接口或者抽象类

    • 一般采用多态的手法去实例化（用变量来保存该匿名内部类的实体）

三、代码块

  • { }

  • 位置区别

    • 构造代码块（类中方法外）

      • 在创建对象时调用
      • 优先于构造方法执行

    • 局部代码块（方法内）

      • 限定变量生命周期，提高内存利用率

    • 静态代码块，在代码块前加static

      • 在创建对象时调用
      • 优先与构造代码块（优先级最高）
      • 只执行一次

      比较常用，加载数据，加载驱动

      媒介：系统软件和硬件的连接者

四、包

1.什么是包

  • 文件夹
    • 多个class文件进行管理
    • 增加文件夹
  • 命名
    • 公司域名倒写
      • baidu.com
      • 包名：com.baidu.add
