private关键字和get/set方法

发表于 2021-08-25 分类于 Java Valine：
本文字数： 851字     阅读时长 : 1 分钟
了解Java中private关键字的用法；自定义类中怎么使用get/set方法

一、权限修饰符—Private

1.作用：

  • private所修饰的内容，使用的时候，只能在本类中使用

  • private：私有的

    • 只能在本类中，操作private修饰的内容

2.开发中，所有的成员变量都要加private

  • 逻辑使用比较灵活
衍生问题：set/get方法

二、get / set 方法

  • 赋值(set方法)：成员变量加上private后，其他方法（比如main方法）不能给成员变量赋值

    • 但是自己的方法能访问自己的私有成员变量，其他类可以访问本类的成员方法
      • 所以用其他类访问成员方法，成员方法给成员变量修改值

  • 取值(get方法)：创建成员方法，通过return带回返回值给调用方法

    • 使用return原因：更加灵活，数据可以重复利用

3.private修饰成员变量细节优化

  • 如果没有特殊的需求，在给成员变量赋值的时候，直接赋值即可
  • 给访问private修饰的成员方法命名规范
    • set变量名
      • 变量名首字母大写
    • get变量名
      • 变量名首字母大写
作业：
  • 定义学生类
    • 有参和空参构造
    • private
    • 学号
    • 姓名
  • 添加到数组中，并做打印
思路：

  • 创建项目并进行测试

  • 需求分析

  • 创建自定义的学生类

    • class Student

    • 属性 = 成员变量

      • 定义成员变量
        • private int id;
        • private String name;

    • 定义无参、有参构造方法

    • 定义相应的get/set方法

  • 创建存储学生的数组方法

    • public static Student[] init(){}
    • return返回创建的数组

  • 创建遍历数组学生对象的方法

    • 用到get方法，取出学生对象的属性

  • 在main方法中

    • 第一步：调用init获取创建的数组

    • 第二步：创建Student的无参对象，带参对象

      • 无参：通过.set()方法给属性赋值
      • 带参：直接在初始化值时赋值

    • 第三步：将创建的学生对象存入数组

    • 第四步：调用遍历数组学生对象的方法

三、Java的编程规范

  • 见名知意
  • 确保功能的唯一性
  • 确保逻辑的灵活性
  • 规则和代码没有关系。规则语法

2.可变参

  • 格式：数据类型 ... 变量名

  • 固定参和可变参在一起时，可变参必须写在末尾
