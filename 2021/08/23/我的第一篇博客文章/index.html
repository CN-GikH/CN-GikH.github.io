学习反射，包括Class类、Constructor类、Field类、Method类等

一、反射

什么是反射？

  • 利用代码区中的数据（.class文件），去创建对象；省略new

    • 构造方法
    • 成员变量
    • 成员方法

  • 在代码区中存储的class并不是真正的class文件(物理)，而是一个虚拟的对象

    • Class类

二、Class

1.概述

  • 泛型
    • class类型很多，不固定
  • 修饰符：final
    • 不能被继承
  • Class的实例表示正在运行的类和接口
  • 管理class文件对象

2.构造方法

无

3.成员方法

  • 获取class文件对象的三种方式

    • ```java
      方式一：通过Object类中的getClass()方法

    • ```java
      方式三：forName(“String s”)

  • className : 类的全路径

  • static 静态的，可以通过类名调用

  • Class<?>泛型

4.Class文件的内部成员

  • 构造方法：Constructor
  • 成员方法：Method
  • 成员变量：Filed

三、Constructor

1.获取Class构造方法

Ⅰ.getConstructors()

  • ```java
    Constructor<?>[] getConstructors()
    //返回一个包含 Constructor对象的数组， Constructor对象反映了由该 Class对象表示的类的所有公共构造函数。

    • 获取所有的构造方法

Ⅲ.getConstructor()

  • ```java
    Constructor getConstructor(Class<?>… parameterTypes)
    //返回一个 Constructor对象，该对象反映由该 Class对象表示的类的指定公共构造函数。

    • 获取指定的构造方法，private修饰的也可以获取

2.利用构造方法创建Class实例

Ⅰ.newInstance(Object… initargs)

  • ```java
    T newInstance(Object… initargs)
    //使用由此 Constructor对象表示的构造函数，使用指定的初始化参数来创建和初始化构造函数的声明类的新实例。

  • 取出所有Public修饰的成员变量

Ⅱ.getDeclaredFields()
  • 取出所有成员变量，包括private修饰的
Ⅲ.getField(String name)
  • 取出指定的public修饰的成员变量
Ⅳ.getDeclaredField(String name)
  • 获取指定的成员变量，private也可以取出

4.给Class成员变量赋值

Ⅰ.set(Object obj, Object value)
  • 获取公共的成员变量

  • 获取私有的成员变量
