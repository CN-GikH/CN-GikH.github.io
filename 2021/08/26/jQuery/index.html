jQuery

发表于 2021-08-26 分类于 jQuery
本文字数： 1.1k字     阅读时长 : 2 分钟
有关JS中计时器的补充知识点，以及jQuery的学习使用，jQuery 是一个 JavaScript 库,核心是选择器

一、计时器

1. setTimeout()

  • 用于在指定的毫秒数后调用函数（只调用一次）
  • code：必需，要调用的函数或代码
  • millisec：必需，在执行代码前等待的毫秒数

2. setInterval()

  • 方法可按照指定的周期（以毫秒计）来调用函数或计算表达式
  • code：必需，要调用的函数或代码
  • millisec：必须，周期性执行或调用代码之间的间隔

3.补充知识

  • onload()事件，当页面或图像被加载完成后触发事件

二、Jquery

1.概念

  • jQuery 是一个 JavaScript 库
  • 导入Jquery

2.jQuery语法

  • 基础语法 $(selector).action()

    • jQuery的核心就是选择器

3.基本选择器

  • id选择器
  • 类选择器
  • 元素选择器
  • *

4.层级选择器

Ⅰ. ancestor descendant

  • 即后代选择器，忽略层级

  • 在给定的祖先元素下匹配所有的后代元素

Ⅱ. parent > child
  • 即子代选择器，只负责子级
Ⅲ. prev + next
  • 相邻元素选择器，下一个紧邻的兄弟
Ⅳ. prev ~ siblings
  • 同辈选择器，下面的所有兄弟

其他选择器有：基本筛选器、内容选择器，可见性选择器、属性选择器、子元素选择器、表单选择器、表单对象选择器等等

详细使用请查阅JqueryAPI

5.基本筛选器

Ⅰ. :first
  • 获取匹配的第一个元素
Ⅱ. :not(selector)
  • 去除所有与给定选择器匹配的元素
  • selector：用于筛选的选择器

Ⅲ. :even

  • 匹配所有索引值为偶数的元素，从 0 开始计数

Ⅳ. :odd
  • 匹配所有索引值为奇数的元素，从 0 开始计数

Ⅴ.:last()
  • 获取最后一个元素

Ⅵ. :gt(index)
  • 匹配所有大于给定索引值的元素

Ⅶ. :lt(index)
  • 匹配所有小于给定索引值的元素

Ⅷ. :eq(index)
  • 匹配一个给定索引值的元素

Ⅸ. :header
  • 匹配如 h1, h2, h3之类的标题元素

Ⅹ.:lang(language)

  • 选择指定语言的所有元素

  • :lang选择器,匹配有一个语言值等于所提供的语言代码，或以提供的语言代码开始，后面马上跟一个“ - ”的元素。例如，选择器$(“div:lang(en)”)将匹配<div lang="en"> and <div lang="en-us">（和他们的后代<div>），但不包括<div lang="fr">

  • 对于HTML元素，语言值由lang属性决定，也可能由来自meta元素或HTTP头信息决定。

其他：

  • :animated 匹配所有正在执行动画效果的元素

  • :focus 匹配当前获取焦点的元素

  • :target 选择由文档URI的格式化识别码表示的目标元素。

  • :root 选择该文档的根元素。

6. html属性

Ⅰ.html([val|fn])
  • $中的.html() ===》获取值
  • 在一个 HTML 文档中, 我们可以使用 .html() 方法来获取任意一个元素的内容。 如果选择器匹配多于一个的元素，那么只有第一个匹配元素的 HTML 内容会被获取。
Ⅱ. text([val|fn])
  • 取得所有匹配元素的内容
  • 结果是由所有匹配元素包含的文本内容组合起来的文本。这个方法对HTML和XML文档都有效
Ⅲ. val([val|fn|arr])
  • 获得匹配元素的当前值。
  • 在 jQuery 1.2 中,可以返回任意元素的值了。包括select。如果多选，将返回一个数组，其包含所选的值。

7.css属性

  • css(name|pro|[,val|fn])
  • name：要访问的属性名称
  • properties：一个或多个CSS属性组成的一个数组
  • | 表示或
  • name,value键值对形式

三、jQuery事件

1.概念

  • 与Js的事件基本相同，但前缀不带on

2.事件

Ⅰ. click()
  • 触发、或将函数绑定到指定元素的 click 事件
